# 정원 사계절 에셋 가이드

> 2026-07-19 확정. 계절별 에셋 세트의 정본 명세 — 아이템 구성·팔레트·제작 규칙·생성 프롬프트.
> 구조 원칙: **슬롯 10개 + 배경 1장은 사계절 공통, 그림만 계절 세트로 교체** (언락 순서·개수 동일).
> 배치 좌표·움직임은 계절 세트마다 코드(src/lib/garden.js)에서 따로 정의 가능.

## 2. 계절 팔레트 (정체성)

| 계절 | 색 계열 | 인상 |
|---|---|---|
| 봄 | 연분홍·연두·살구빛 하늘 | 깨어남, 몽글 |
| 여름 | **짙은 여름 그린·맑은 하늘색·해바라기 옐로·수국 블루** | 생기, 시원한 아침 |
| 가을 | 주황·갈색·억새 베이지·노을 하늘 | 결실, 따뜻한 쓸쓸함 |
| 겨울 | 흰색·회청 + 동백·열매의 빨강 포인트 | 고요, 포근한 정적 |

- 공통 톤: **성인 취향 — 채도는 중간(뮤티드와 비비드의 중간), 원색 금지, 무광**
- 1차(원색·유아틱)와 2차(회색빛·우울)의 실패를 반복하지 않기: "계절 정체성은 채도가 아니라 모티프·색 계열로"

## 3. 제작 공통 규칙

- 무광 클레이(펠트) 질감 + 은은한 종이 그레인, 만화 외곽선·광택·귀여운 얼굴(큰 눈·볼터치) 금지
- **빛은 오른쪽 위** (해 위치 기준), **자체 음영만** — 바닥 그림자는 코드가 처리하므로 절대 그리지 않기
- 바닥 요소(식물·동물·눈사람)는 **밑동이 캔버스 맨 아래에 닿게**, 아래 여백 없이
- 군락(꽃·새싹 등)은 가운데 크고 좌우 작게, 키 서로 다르게 — **키 낮고 옆으로 퍼진 반원형 실루엣**, 캔버스 90% 이상 채우기
- **배경은 순수한 마젠타(#FF00FF) 단색으로 생성** (투명 내보내기 금지 — 크림색 배경으로 인한 알파 손상 사고 재발 방지, 제거는 코드가 수행)
- 낱개 512×512 이상 정사각 PNG (연못류는 가로 2:1), 배경은 1080×810
- 낱개 요소를 배경에 그려 넣지 않기 (각자 움직여야 함)

## 4. 파일 네이밍 (계절 접두사)

`garden-{season}-{slot}.png` — season: spring | summer | autumn | winter
slot: sprout · cloud · flower-a · special(나비/잠자리/눈사람) · tree-small · sun · flower-b · animal · pond · tree-big
배경: `garden-{season}-bg.png` (앱 적용 시 JPG 변환)

※ 현행 여름 에셋은 무접두사(garden-*.png) — 여름 재생성 세트부터 접두사 체계로 전환하고 코드도 함께 갱신.

## 5. 지금 할 일 — 여름 세트 재생성 7장

재생성: 배경 · 큰 나무 · 작은 나무 · 수국 군락(꽃A 교체) · 해바라기 군락(꽃B 교체) · 새싹 3모음 · 뭉게구름
유지: 해 · 새 · 나비 · 연못 (계절 중립 — 새 배경 위에서 어색하면 코드 보정)

### 공통 스타일 문단 (모든 생성 앞에 붙이기)

```
성인 취향의 차분한 힐링 일러스트. 부드러운 무광 클레이(펠트) 질감, 은은한 종이 그레인.
계절은 한여름 아침 — 생기 있는 여름 색이지만 원색은 금지, 채도는 중간(뮤티드와 비비드의 중간).
팔레트: 짙은 여름 그린(녹음), 프레시 그린, 맑은 하늘색, 해바라기 옐로, 수국 블루·라벤더, 웜 크림.
빛은 오른쪽 위에서 부드럽게. 물체 자체의 음영만 넣고 바닥 그림자는 절대 그리지 않기.
귀여운 얼굴·눈 없음, 만화 외곽선 없음, 광택·반짝임 없음.
배경은 순수한 마젠타 단색(#FF00FF). 정사각형 512×512 이상 PNG.
```

### 개별 프롬프트

**배경 garden-summer-bg** (이것만 마젠타 불필요, 1080×810):
```
여름 아침의 언덕 풍경 배경. 위 55%는 맑고 부드러운 여름 하늘색(위로 갈수록 살짝 진하게),
아래 45%는 생기 있는 여름 풀밭 초록(짙은 그린~프레시 그린), 완만한 곡선 지평선.
수채 종이 질감. 다른 사물 없이 하늘과 풀밭만.
```

**큰 나무 / 작은 나무**:
```
한여름의 잎이 무성한 활엽수. 기존 나무 참조 이미지와 같은 실루엣·스타일 유지.
잎은 짙고 풍성한 여름 녹음(깊은 초록, 회색빛 금지), 줄기는 따뜻한 갈색.
밑동은 캔버스 맨 아래에 닿게. (큰 나무·작은 나무 각각 생성)
```

**수국 군락 (꽃A 교체)**:
```
수국 세 송이 군락. 둥근 공 모양의 수국 꽃송이, 블루·라벤더 계열의 차분한 색.
가운데 송이가 가장 크고, 좌우 송이는 작고 키가 서로 다르게.
짙은 그린 잎이 아래에서 덤불처럼 받치고, 밑동은 캔버스 맨 아래에 닿게.
키 낮고 옆으로 퍼진 반원형 실루엣, 캔버스 90% 이상 채우기.
```

**해바라기 군락 (꽃B 교체)**:
```
밝은 해바라기 옐로(원색 아님)의 작은 해바라기 세 송이 군락.
가운데 송이가 가장 크고 정면, 좌우는 작고 살짝 옆을 향하게, 키 서로 다르게.
짙은 그린 줄기·잎이 아래에서 덤불처럼 모이고, 밑동은 캔버스 맨 아래에 닿게.
키 낮고 옆으로 퍼진 반원형 실루엣, 캔버스 90% 이상 채우기.
```

**새싹 3개 모음**:
```
프레시 그린 새싹 세 개가 옹기종기 모여 난 작은 무리.
가운데가 가장 크고 떡잎 두 장, 양옆은 더 작고 키가 서로 다르게 — 자연스러운 삼각 실루엣.
세 줄기 밑동이 모두 캔버스 맨 아래에 닿게. 키 낮고 옆으로 퍼진 형태, 캔버스 90% 이상 채우기.
```

**뭉게구름**:
```
여름 하늘의 뭉게구름 한 덩어리, 뭉실뭉실 볼륨감 있게.
흰색과 아주 연한 웜그레이 음영, 무광. 기존 구름 참조 실루엣 유지.
```

### 업로드 전 셀프 체크
- [ ] 여름 신호가 보이는가 (짙은 녹음·맑은 하늘·수국·해바라기)
- [ ] 원색·광택·얼굴 없음, 무광 클레이 질감인가
- [ ] 바닥 요소 밑동이 캔버스 바닥에 닿아 있는가
- [ ] 배경이 마젠타 단색인가 (배경 제외)

## 6. 애니메이션·그림자 (코드 담당, 참고)
