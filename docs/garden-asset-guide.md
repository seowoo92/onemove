# 여름 정원 그림 작업 가이드

> 2026-07-19 확정 기획 기준. 정원 탭(게이미피케이션)에 들어갈 일러스트 명세.
> 기획 요약: 계절제 정원 — 루틴 완료가 쌓이면 요소가 하나씩 열려 정해진 자리에 자동 배치되고,
> 계절이 끝나면 완성된 정원이 "사계절 앨범"에 저장된다. 요소는 화면을 보는 동안 계속 살아 움직인다.

## 스타일 공통

- 기존 에셋(코치 캐릭터·마음 날씨 아이콘·완료 배너)과 같은 **클레이 질감·부드러운 음영** 톤
- 앱 팔레트와 어울리게: 크림 #FAF6F0 · 딥그린 #24523F · 코랄 #EE8466 · 옐로 #F3D978 · 하늘색 #A9CFE0
- 외곽선 없는 말랑한 형태, 과한 디테일보다 멀리서 봐도 예쁜 실루엣 위주
- **요소는 전부 낱개 그림**입니다 — 배경에 그려 넣지 마세요 (코드가 각자 움직여야 해서)

### 그림자 규칙 (2026-07-19 확정)
- **에셋에는 자체 음영만** (몸체의 밝은 면/어두운 면). **바닥 그림자는 그리지 않는다** — 접지 그림자는 코드가 일괄 처리 (몸체가 살랑여도 그림자는 바닥에 고정되는 연출을 위해)
- **빛 방향 통일**: 햇님이 오른쪽 위(78%, 14%)에 있으므로 모든 요소의 밝은 면은 **오른쪽 위**
- **바닥 요소(새싹·꽃·나무)는 캔버스 맨 아래에 밑동이 닿게** — 아래 여백 최소화 (코드 그림자와 밑동 정렬용)
- 밑동 끝을 살짝 어둡게 누르면 접지감이 더 좋아짐 (선택)
- 코드 처리 계획: 바닥 요소=납작 타원 그림자 고정 / 하늘 요소(구름·햇님)=없음 / 나비=높이에 따라 작아지는 옅은 그림자 / 연못=그림자 없음(수면 반짝임)
