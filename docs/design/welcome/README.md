## 파일
- `웰컴 진입화면.dc.html` — 웰컴 진입화면 (390×812)
- `support.js` — 런타임 (브라우저에서 바로 열기용)

## 여는 법
`웰컴 진입화면.dc.html`를 브라우저에서 더블클릭하면 바로 렌더됩니다. (support.js 동봉)

## 사양
- **사이즈:** 390 × 812 (iPhone 기준), 모서리 라운드 46px
- **배경:** 화면 #F9F6EE / 바깥 #E6E1D6
- **브랜드 컬러:** 딥그린 #24523F · 코랄 #EFA58F · 옐로 #F3D978
- **폰트:** Pretendard Variable (CDN)
- **모티프:** 점이 좌하→우상으로 커지며 올라가는 "오르막 한 걸음" (그린→코랄→옐로→딥그린 도착점)
- **카피**
  - 헤드라인: 오늘 / 할 수 있는 / 만큼만.
  - 서브: 오늘의 한 걸음, 그거면 충분해요. / 완벽 대신 꾸준함으로 멈춘 자리에서 다시 이어가요.
- **CTA:** 카카오로 시작하기(#FEE500) · 로그인 없이 시작하기

## 직전 수정 반영
