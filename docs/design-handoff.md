# 핸드오프: 오늘만큼 (onemove) — 화면 리디자인

## 개요
무기력한 날에도 "오늘 할 수 있는 만큼"만 제안하는 CBT 기반 회복탄력성 AI 루틴 코치 웹앱.
이 패키지는 **전체 화면 흐름의 고해상도(hifi) 디자인 시안**과 디자인 시스템 정의를 담고 있습니다.

- 배포: https://seowoo92.github.io/onemove/
- 스택: Vite + React 19 + Tailwind CSS v4, Supabase Auth(카카오 OAuth), Upstage Solar API
- MVP 발표: 2026-06-22

## 이 디자인 파일들에 대하여
번들 안의 `.dc.html` 파일은 **HTML로 만든 디자인 레퍼런스(시안)** 입니다 — 그대로 가져다 쓰는 프로덕션 코드가 아니라, 의도한 모양·동작을 보여주는 프로토타입입니다.
할 일은 이 시안들을 **기존 코드베이스 환경(React 19 + Tailwind v4)** 에서, 그 프로젝트의 기존 컴포넌트·패턴으로 **다시 구현(recreate)** 하는 것입니다. 모든 색·치수·타이포는 아래에 정확히 적어두었으니 시안 픽셀을 그대로 재현하면 됩니다.

## 충실도
**고해상도(hifi)** — 최종 색상, 타이포, 간격, 인터랙션이 모두 확정된 시안입니다. 기존 Tailwind 토큰으로 픽셀 단위까지 동일하게 구현하세요.

## 화면 (시안보드 A~F 구역에 대응)

> 전체 흐름: **S0 진입 → S1 코치 선택 → S2 마음 날씨 → S3 루틴 홈 → S4 코치 메시지(모달)**. 하단 탭바: 홈 / 기록 / 설정.

### A. 온보딩
1. **01 웰컴/진입** — 로고 + 히어로 "오늘 / 할 수 있는 / 만큼만." + 안내문. 하단 고정: 카카오 시작 버튼(노랑) + "로그인 없이 시작하기" 텍스트 버튼. 하단 장식 클레이 풍경(파스텔 원형). 최초 사용자만 표시.
2. **02 코치 선택** — 큰 클레이 원(현재 선택 코치 150px) + 코치명 + 한마디. 하단 3개 썸네일(유쾌/진중/다정, 56px, 선택 시 `box-shadow:0 0 0 3px #24523F` 링) + 1차 버튼 "OO로 시작".
3. **03 마음 날씨** — 좌상단 뒤로 버튼(42px 흰 카드). 제목 "오늘 마음 날씨는 어떤가요?" + 날짜. 3개 행 카드(좋아요/맑음, 보통이에요/구름, 힘들어요/비) — 각 54px 날씨 아이콘 + 제목 + 부제. 하단 도움말 말풍선(`?` 아바타 + 2개 버블).

### B. 홈 — 오늘의 루틴 (스크롤)
- 헤더: "OO님, 오늘도 와줘서 고마워요" + "6월 18일 (목) · 힘들어요"
- 진행 카드(딥그린): "오늘 1 / 3개 완료" + 우측 "한 걸음씩"(노랑) + 진행 바(`linear-gradient(90deg,#D9F2EE,#F3D978)`)
- 완료된 루틴 행: 연녹(`#EFF4EE`) 배경 + 체크 원(딥그린) + 취소선 텍스트 + "완료"(`#6FB988`)
- 루틴 카드(흰색): 루틴명 + 난이도칩 + 영역칩 + 한 줄 설명 + 버튼 2개 [완료(노랑)] [지금은 어려워요(회색)]
- 쉬운버전 전환 카드: 상단 "더 쉬운 버전으로 바꿨어요" 배지(`#FBEDE6`), 원래명 취소선 + 새 루틴명, 버튼 [완료] [오늘은 쉬어가기(코랄톤)]
- 카카오 공유 버튼(노랑) + 안내 캡션
- 하단 탭바(홈 활성)

### C. 기록 (스크롤)
- 제목 "기록" + "최근 14일 · 멈춘 날도 기록이에요"
- 14일 막대 차트(마음 날씨별 색): 좋아요 `#F3D978` / 보통 `#9FD2B0` / 힘들어요 `#A9CFE0` + 범례
- 날짜별 타임라인: 좌측 세로선 + 상태색 점, 날짜 + 상태/날씨 + "N / M" + 완료 루틴 칩들

### D. 설정 (스크롤)
- 제목 "설정"
- 계정 카드: 닉네임 "OO님" + 변경 버튼 + 코치명 + 카카오 연결 상태 버튼(노랑)
- 섹션(라벨 11.5px `#9AA69D`): **알림**(카톡 알림 토글, on=`#24523F`) / **AI 코치 성격**(3개 카드, 선택 시 `0 0 0 2.5px #24523F`) / **마음 날씨**("다시 고르기" 행 + 경고 부제) / **마음이 많이 힘들 때**(위기상담 2카드: 1577-0199, 109)
- 하단 탭바(설정 활성)

### E. 오버레이
- **07 AI 코치 메시지 모달**: 바텀시트(`#F9F6EE`, 상단 34px 라운드). 핸들바 + 코치 아바타(52px) + 코치명/"코치가 한마디 건네요" + 메시지 본문(19px, min-height 90px) + 우측 "출처 · AI" 배지 + "계속하기" 1차 버튼. 배경 딤 `rgba(20,46,34,.45)`. 로딩 중엔 "코치가 메시지를 작성하고 있어요…" + 점 3개 + 스켈레톤.
- **08 카카오 공유 시트**: 바텀시트 + 미리보기 카드(상단 클레이 풍경 SVG 124px + 흰 본문: 로고/날짜, "오늘 할 수 있는 만큼, N개 완료", 체크리스트, 푸터 문구) + 하단 [닫기] [카카오톡으로 보내기(노랑)]
- **09 전송 완료 토스트**: 딥그린 알약, 카카오 아이콘(노랑) + "내 카카오톡으로 보냈어요"
- **완료 축하 카드**(전체 완료 시): 상단 그라데이션 헤더 + 해 반원, "오늘 하루도 잘 해냈어요" + 위로 문구 + 코치 칩

### F. 아이콘 & 요소 (전부 CSS/SVG, 외부 이미지 아님)
- **브랜드 마크**: 겹친 3개 원(코랄 작은 점 + 옐로 중간 점 + 딥그린 큰 점, `margin-left:-6px`로 겹침). 로고 옆 워드마크 "오늘만큼" 800.
- **코치 캐릭터**: 위 클레이 셰이딩 원 3종(유쾌/진중/다정)
- **앱 아이콘·파비콘**: `assets/app-icon-512.png`, `favicon-180.png`, `favicon-32.png` (제공됨 — 아래 Assets 참고)
- **마음 날씨 아이콘 3종**(맑음/구름/비), **탭 아이콘 3종**(홈/기록/설정 — stroke 1.8 라인 아이콘), 카카오·뒤로 버튼
- **상태 배지·칩**: 완료 체크 원, 쉬어가기(일시정지), 쉬운버전 별 배지, 난이도칩(아주 쉬움/쉬움/보통), 영역칩, 진행률 바

> **아이콘 원칙**: 이모지 절대 금지. 아이콘은 stroke 라인 SVG 또는 CSS 도형으로만. 텍스트 중심 UI.

## 인터랙션 & 동작
- 코치 썸네일/카드 탭 → 선택 링 표시, 미리보기 갱신
- "지금은 어려워요" → 해당 카드가 쉬운버전으로 인플레이스 전환(배지 추가, 버튼이 [완료][오늘은 쉬어가기]로)
- 완료/어려워요/쉬어가기 선택 → S4 코치 메시지 모달(바텀시트 슬라이드업)
- 카카오 공유 → 공유 시트 → 전송 후 토스트(약 2초)
- 토글/탭 활성색 전환은 즉시
- 반응형 3모드: 모바일(<480px) 전체폭 / 태블릿(480~1023px) 480px 중앙+그림자 / 데스크톱(≥1024px) 좌측 소개패널 + 우측 폰 베젤(390×844, radius 52px, 테두리 `#1C3F2F`)
