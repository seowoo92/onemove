# 오늘만큼 (onemove) — 디자인 구현 가이드

---

## 1. 문서 목적 · 구현 접근

이 문서는 오늘만큼 화면 구현의 기준 명세입니다. 1차 시안(디자인 핸드오프 패키지) 기준.

**목표는 픽셀 완전 재현이 아닙니다.** 정해진 색 토큰·타이포·간격·컴포넌트 규칙을 지켜서 안정적으로 구현하는 것이 목표입니다.

- 기능 로직은 기존 코드(React 19 + Tailwind CSS v4)를 그대로 유지
- 이미지 에셋이 없는 항목은 CSS/SVG 대체 요소를 먼저 구현하고, 에셋이 준비되면 단계적으로 교체
- 시안 `.dc.html` 파일은 디자인 레퍼런스(프로토타입)이지 그대로 쓰는 코드가 아님. 색·치수·타이포 확인용으로만 활용

---

## 2. 현재 진행 상태 (DONE — 건드리지 말 것)

아래는 이미 구현·배포 완료된 기능입니다. Claude Code는 이 항목들을 새로 만들거나 되돌리지 말 것.

- 온보딩(WelcomeScreen) / 코치 선택 / 마음 날씨 선택(StateCheck)
- 홈(루틴 목록, 완료, "지금은 어려워요" → 쉬운 버전 전환, N/M 완료 진행 바)
- AI 코치 메시지 모달(CoachModal — 하단 시트 구조) + 예비 메시지 폴백
- 기록 화면(RecordScreen): recharts 14일 막대 그래프 + 타임라인 (**이미 완성·배포됨, "나중에"가 아님**)
- 설정(SettingsScreen): 닉네임, 카카오톡 알림 토글, 코치 변경, 마음 날씨 다시 고르기, 위기 상담 연락처
- 카카오 로그인 (Supabase OAuth, 작동 중)
- localStorage 유지, 반응형 3모드(모바일/태블릿/데스크톱 베젤), Pretendard 폰트, 파비콘·앱 아이콘

**현재 국면은 "기능 구현"이 아니라 "디자인 적용 + 이미지 교체"입니다.**

## 6. 화면 사양 (S0~S4 + 탭)

> 전체 흐름: S0 진입 → S1 코치 선택 → S2 마음 날씨 → S3 루틴 홈 → S4 코치 메시지(모달). 하단 탭바: 홈 / 기록 / 설정.

### S0 — 웰컴 (오르막 모티프, 기존 웰컴 대체)

구성 (위→아래):
1. **브랜드 마크** — 매트 클레이 3점(코랄 `#EFA58F` · 옐로 `#F3D978` · 딥그린 `#24523F` 겹침) + 워드마크 "오늘만큼" (800)
2. **헤드라인** — "오늘 / 할 수 있는 / 만큼만." (39px / 800 / letter-spacing -0.045em / line-height 1.22 / color `#24523F`)
3. **서브카피** — "오늘의 한 걸음, 그거면 충분해요. / 완벽 대신 꾸준함으로 멈춘 자리에서 다시 이어가요." (15px / 500 / `#6F7D72` / line-height 1.7)
4. **오르막 점 패스 (SVG)** — 점 5개가 좌하→우상으로 커지며 올라감. 색 순서: 연녹 `#CFE0D4` → 녹 `#9FC1A9` → 코랄 `#EFA58F` → 옐로 `#F3D978` → 딥그린 `#24523F`(도착점). 점들 사이 점선(dashed) 연결. "한 걸음씩 나아간다"는 방향성 표현. **전부 CSS/SVG로 구현, 이미지 불필요.**
5. **CTA** — "카카오로 시작하기" (`#FEE500`, 카카오 아이콘) + "로그인 없이 시작하기" (텍스트 버튼)

구현 주의:
- 첫 사용자(코치 미선택)에게만 표시
- 시안 `.dc.html`의 가짜 상태바(9:41)·폰 프레임은 목업 장식 — 구현하지 말 것
- 절대좌표(812px 기준 `absolute`/`bottom` 고정) 대신 반응형 flex로 배치 (모바일/태블릿/데스크톱 베젤 모두 대응)
- 카카오 버튼 위 혜택 안내 캡션 → 8월 보내기 기능과 연계, 지금은 보류
