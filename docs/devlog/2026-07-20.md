## 2. 사계절 앨범 실저장·열람 UI
- **저장 방식**: 별도 저장 없이 onemove_history에서 지난 계절 계산(getPastSeasons) — 정원과 같은 원칙, 동기화로 복원
- GardenScene 컴포넌트 분리 (본 화면·앨범 공용), SEASON_SETS 구조(계절→에셋 세트, 가을부터 등록만 하면 됨)
- **폴라로이드 앨범 확정**: 연도 페이지 안에 계절 슬롯 — 채워진 계절은 기울어진 폴라로이드(탭→모달 열람), 지금 계절 "가꾸는 중" 배지, 빈 계절 점선(미래 "곧 만나요"/과거 "지나갔어요")
- 반복 조정: 2×2 → **4칸 한 줄**(한 화면 정돈), 올해만 표시, 다음 해 구분선만 표기
- 성장 카드: 요소별 고정 등장 문구(arrival — 조사 오류 해소), 도감 한 줄 10원(전용 중앙정렬 썸네일 자동 생성), 요소명 수국·해바라기 정정

## 3. 모바일 실기기 대응 (아이폰 캡처 피드백)
- iOS 안전영역: viewport-fit=cover + 탭바 하단 env(safe-area-inset-bottom) — 홈 인디케이터 가림 해소
- 상태바·오버스크롤 영역 크림 배경, 상단 여백 축소
- 탭 화면 공통 ScreenHeader 컴포넌트 (정원·기록·설정 타이틀·부제 위치 통일), '나의 정원' 타이틀
- 실화면 스크린샷 도구 확립 (puppeteer 헤드리스 + 온보딩 시딩 — 이후 모든 시안 검수에 활용)
