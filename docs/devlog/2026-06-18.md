### 트러블슈팅 — KOE205 에러
- 증상: 카카오 로그인 시 "설정하지 않은 동의 항목: account_email" 에러
- 원인: Supabase Auth가 account_email 스코프를 서버에서 강제 요청하나, 카카오는 개인 개발자 앱에 이메일 항목을 허용하지 않음. 프론트에서 scope를 profile_nickname만 요청해도 Supabase 서버가 강제 추가하여 코드로는 해결 불가 (Supabase 공식 이슈 #36878)
- 해결: 카카오 비즈앱 전환 + 이메일 동의항목 신청 → 정상 로그인 구현
  - 비즈앱 전환에 앱 아이콘이 필요하여 아이콘 제작과 연계
- account_email은 선택 동의로 유지 (필수 동의 미적용)
