# 개발일지 2026-07-19 (일) — 리눅스 서버 첫 가동 + 탭 4개 개편 + 정원 게이미피케이션 완성

> 리눅스 서버에서의 첫 실작업일. 환경 셋업부터 2차 스프린트 최대 기능(정원)까지 하루에 완료.

## 1. 리눅스 서버 개발 환경 완성

- Node 20.20.2 설치 (nodejs.org 공식 바이너리 → ~/.local, GitHub Actions와 동일 메이저)
- npm ci · .env.local 생성 (anon 키는 미기입 — 게스트 모드로 개발, 배포판은 GitHub Secrets 사용)
- **SSH 키 생성·GitHub 등록** → 푸시·자동 배포 파이프라인 재개통 (이관 후 첫 배포 성공)
- Supabase `profiles.pinned_ids` 컬럼 ALTER 실행 확인 (7/14 대기 항목 해소)

## 2. 탭 4개 개편 (오늘 · 정원 · 기록 · 설정)

- 기존 3탭(홈/기록/설정) → 4탭. "홈"은 "오늘"로, **정원 탭 신규** (새싹 라인 아이콘)
- 계획 변경: 문서상 "코치 탭"이었으나 내용이 빈약해 **정원(게이미피케이션) 탭으로 대체** — 사용자 확정

## 3. 기록 탭 대시보드 승격

- **월간 마음 날씨 캘린더** (MonthCalendar.jsx): 날짜 칸에 날씨 아이콘, ◀▶ 월 이동, 오늘 하이라이트, 월 요약(기록일·완료 수)
- 연속 기록 배지 (2일 이상 "N일 연속 기록 중") — 7/10 멘토링 피드백 '분석 대시보드' 반영

## 4. 하루 마무리 (AI 회고)

- 모든 루틴 정리(완료/쉬어가기) 시 완료 축하 카드에 **하루 종합 2문장** 표시 (generateDailyReview)
- 완료 루틴들의 영역을 묶어 종합 ("몸을 깨우고 바깥 공기도 쐰 하루") + 부드러운 마침 인사. 내일 다짐·권유 금지 (기존 코치 멘트 원칙 통일)
- 전부 쉬어간 날은 수용·안심 톤 분기. 예비 메시지 6종은 "일상 입말" 기준으로 사용자 확정 (AI스러운 표현 배제 피드백 반영)
- 하루 1회 생성 후 onemove_review 재사용 (자정 리셋). Solar 호출·품질 게이트는 requestSolar로 공용화

## 5. 정원 게이미피케이션 (오늘의 최대 작업)

### 기획 확정
- **계절제 + 사계절 앨범**: 계절 누적 완료 수로 요소 10종 언락(2~75개), 계절 종료 시 앨범 갈무리(리셋 아님 — 상실감 방지), 퇴보 없음
- **사계절 세트 설계**: 슬롯 10개 공통 구조, 계절별 아이템 확정 (가을 억새·낙엽 웅덩이, 겨울 눈사람 등) — docs/garden-asset-guide.md 정본
- 데이터는 onemove_history/daily_entries에서 계산 — 신규 저장 구조 없음 (단일 source of truth)

### 에셋 제작 (사용자 생성 × Claude QC 반복)
- 1차(원색·유아틱) → 2차(저채도·우울) → **여름 정체성 상실 발견** → 여름 재생성 라운드
- 시안 3장 선행 검수 방식 도입: 질감 앵커(새 이미지 참조)로 "무광 펠트 클레이" 통일
- 마젠타 배경 생성 → 코드 제거 파이프라인 확립 (1차 크림 배경으로 구름 알파 손상 사고 재발 방지)
- 톤 보정 체인 확정: T1 파스텔(흰색 12%+채도-8) → 배경 채도-10·요소-5 → **대비 +8%** (채도 다운으로 인한 흐릿함을 대비로 회복 — 명도 아닌 대비가 정답)
- 배경 2.4MB → 118KB JPG 경량화, 계절 접두사 네이밍(garden-summer-*) 전환

### 본 화면 구현
- 4:3 장면에 언락 요소 %좌표 자동 배치, **본체·접지 그림자 레이어 분리** (몸체가 흔들려도 그림자는 바닥 고정)
- 상시 애니메이션: 꽃 살랑·나무 호흡·**구름 75초 가로지르기**·햇님 숨쉬기·나비 곡선 비행+날갯짓·새 갸웃·연못 반짝임
- 신규 언락 요소 1회 등장 연출(onemove_garden_seen), prefers-reduced-motion 전체 정지
- `?garden=숫자` 미리보기 파라미터 (발표 데모용)
- 배치·크기 다회 반복 조정 (합성 시안 왕복 검수 방식)

### 마감 후 추가 — 정원 탭 UI 개선 (사용자 피드백)
- 요소 도감: 열린 요소는 원형 썸네일, 안 열린 요소는 점선 물음표 원으로 개수 표시
- 문구 교정: "이번 여름 완료 75개" → "이번 여름에 완료한 루틴 75개" / 다음 요소 안내도 풀어씀
- 사계절 앨범 설명은 물음표 버튼 토글로 접음

## 배운 것 / 다음에 참고
- 이미지 AI 협업: **참조 기준(실루엣/질감)을 파일로 명시**하고, 시안 3장 → 합성 검수 → 잔여 생성 순서가 왕복을 줄임
- 채도·대비·명도는 각각 다른 문제를 푼다: 유아틱=채도, 우울=명도, 흐릿=대비
- 계절 정체성은 채도가 아니라 모티프·색 계열에서 나온다
