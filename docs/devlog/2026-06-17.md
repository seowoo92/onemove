### 버그 수정 (2차)

#### 버그 1: 코치 메시지 반복 출력 (solar.js)
- 원인: Solar API가 구두점 없이 문장을 반복할 경우(`차분합니다차분합니다...`),
  기존 regex `/[^.!?]+[.!?]+/g`가 전체를 단일 문장으로 인식 → 중복 제거 불가
- 수정:
  - `removeRepetition` 함수 추가 — `(.{8,})\1+` → `$1` 로 반복 구절 선제 제거
  - `cleanMessage` 파이프라인에 2번째 단계로 삽입
  - fallback 경로도 `cleanMessage` 거치도록 수정 (기존에는 raw 반환)

#### 버그 2: 홈 탭 버튼 무반응 (App.jsx)
- 원인: 이미 홈 탭에 있을 때 클릭하면 `setActiveTab('home')`이 동일 state → React no-op
- 수정:
  - `homeKey` state 도입, `handleTabChange` 함수에서 홈 탭 클릭 시 항상 증가
  - Home 컴포넌트에 `key={homeKey}` 부여 → 클릭마다 리마운트, 시각적 반응 보장
  - 기록/설정에서 홈으로 돌아오는 경우도 동일하게 처리
